## 设计说明

这些线框图使用ASCII字符绘制，主要目的是展示系统的界面结构和布局，帮助开发团队理解系统的功能和交互流程。线框图不包含具体的样式和视觉设计，仅作为功能和布局的参考。

## 使用指南

1. 线框图使用Markdown格式，可以在任何支持Markdown的查看器中查看
2. 每个文件对应系统的一个主要功能模块
3. 线框图中的方括号`[]`表示按钮，圆括号`()`表示单选按钮，方括号加勾选`[✓]`表示复选框
4. 线框图中的`|`和`-`用于绘制边框和分隔线
