## 权限详细说明

每个功能模块的权限细分为四种操作类型：

| 操作类型 | 描述 |
|---------|------|
| 查看(View) | 允许查看数据，但不能修改 |
| 创建(Create) | 允许创建新数据 |
| 编辑(Edit) | 允许修改现有数据 |
| 删除(Delete) | 允许删除数据 |

例如，对于池塘管理模块，权限可细分为：
- 查看池塘信息(pond.view)
- 创建新池塘(pond.create)
- 编辑池塘信息(pond.edit)
- 删除池塘(pond.delete)

租户管理员可以针对每个角色精确配置这些细粒度的权限，实现更精细化的权限控制。
