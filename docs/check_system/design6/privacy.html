自律打卡应用隐私政策

本隐私政策描述了我们如何收集、使用和保护您在使用自律打卡应用（以下简称"本应用"）时提供的个人信息。我们重视您的隐私，请您在使用本应用前仔细阅读本隐私政策。

1. 我们收集的信息

1.1 您主动提供的信息：包括但不限于注册账号时提供的用户名、密码、邮箱地址、手机号码等个人信息。

1.2 使用信息：我们会自动收集您使用本应用的相关信息，如任务完成情况、打卡记录、使用频率等数据。

1.3 设备信息：我们可能会收集您使用的设备信息，如设备型号、操作系统版本、设备标识符、IP地址等。

2. 信息的使用

我们使用收集到的信息主要用于以下目的：

  • 提供、维护和改进本应用的功能与服务
  • 处理您的账号注册和登录需求
  • 记录并展示您的打卡数据和统计信息
  • 提供个性化的用户体验和内容推荐
  • 发送服务通知和更新信息
  • 分析用户行为以优化产品功能和用户体验
  • 预防和处理欺诈、安全风险等问题

3. 信息的共享与披露

除以下情况外，我们不会与任何第三方分享您的个人信息：

  • 在获得您明确同意的情况下
  • 为提供服务必须与合作伙伴共享的必要信息
  • 遵循法律法规要求、法律程序或政府强制性要求
  • 在涉及合并、收购、资产转让或类似交易时，如涉及到个人信息转让，我们会要求新持有您个人信息的公司或组织继续受本隐私政策约束

4. 信息安全

我们采取合理的安全措施保护您的个人信息，包括但不限于数据加密、访问控制等技术手段和管理制度，防止您的个人信息泄露、损毁或丢失。

5. 信息存储

我们收集的个人信息将存储在中华人民共和国境内的服务器上。我们会在法律法规要求的期限内，或为实现本政策所述目的所必需的期限内保留您的个人信息。

6. 您的权利

根据适用的法律法规，您可能享有以下权利：

  • 访问、更正或删除您的个人信息
  • 撤回您的同意
  • 注销您的账号
  • 获取您个人信息的副本
  • 对我们处理您个人信息的方式提出异议

如需行使上述权利，请通过本政策末尾提供的联系方式与我们联系。

7. 儿童隐私

本应用不面向13岁以下的儿童。我们不会故意收集13岁以下儿童的个人信息。如果您是父母或监护人，发现您的孩子未经您同意向我们提供了个人信息，请联系我们，我们将采取措施删除相关信息。

8. 隐私政策的更新

我们可能会不时更新本隐私政策。当我们对隐私政策做出重大变更时，我们会在应用内发布通知，并在政策生效前取得您的同意。建议您定期查看本政策，了解我们如何保护您的个人信息。

9. 联系我们

如您对本隐私政策有任何疑问、意见或建议，或者希望行使您的数据权利，请通过以下方式联系我们：

电子邮件：privacy@selfdiscipline-app.com
