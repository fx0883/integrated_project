自律打卡应用

扁平化设计 响应式布局 移动优先

帮助用户养成良好习惯，提高自律能力

本项目展示了自律打卡应用的扁平化界面设计。扁平化设计摒弃了拟物化的视觉元素，采用简洁的色彩、干净的排版和简单的界面元素，让用户能够更专注于应用的核心功能。
