子账户管理
什么是子账户？
子账户功能可以帮助您为家人创建单独的任务和记录，适合父母为孩子管理学习任务、夫妻共同打卡健身等场景。每个主账户最多可以创建3个子账户。

使用说明
  1. 每个主账户最多可创建3个子账户
  2. 切换到子账户后，所有任务和打卡记录都属于该子账户
  3. 主账户可以查看和管理所有子账户的数据
  4. 删除子账户将永久删除其所有数据，请谨慎操作
  5. 可以在设置中开启同步提醒，当子账户完成打卡时主账户也会收到通知
