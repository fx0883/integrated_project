常见问题
如何创建新任务？
点击任务页面底部的"+"按钮，然后填写任务名称、设置时间和重复规则，最后点击保存即可。
如何修改任务提醒时间？
在任务详情页面，点击"编辑"按钮，然后您可以调整提醒时间和频率，完成后点击保存。
如何导出打卡数据？
在"我的"页面点击设置，找到"数据管理"选项，然后选择"导出数据"，您可以选择导出的时间范围和格式。
如何添加和管理子账户？
在"我的"页面找到"子账户管理"选项，点击进入后可以添加、删除或切换子账户。子账户适合为家人创建单独的任务和记录。
应用闪退怎么办？
请尝试以下方法：1. 更新到最新版本 2. 清除应用缓存 3. 重启设备。如果问题仍然存在，请通过上方表单提交详细问题，务必提供您的设备型号和系统版本，我们会进一步协助您解决。
