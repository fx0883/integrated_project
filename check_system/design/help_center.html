常见问题

如何创建新的打卡任务？

创建新的打卡任务非常简单：

  1. 在底部导航栏点击"+"按钮
  2. 或在任务列表页面点击右上角的"+"图标
  3. 填写任务名称、选择打卡类型
  4. 设置开始日期和提醒时间
  5. 点击"创建任务"按钮完成

您还可以从预设模板中选择，快速创建常用任务。

如何修改或删除已创建的打卡任务？

修改或删除已创建的打卡任务：

  1. 在任务列表页面，长按需要操作的任务
  2. 在弹出的菜单中选择"编辑"或"删除"
  3. 如果选择编辑，可以修改任务的各项内容
  4. 完成编辑后点击"保存"按钮

注意：已经完成的历史打卡记录不会因为删除任务而消失，您仍然可以在统计页面查看历史数据。

如何查看我的打卡统计和历史记录？

查看打卡统计和历史记录的方法：

  1. 点击底部导航栏的"统计"图标
  2. 在统计页面可以看到您的整体打卡情况
  3. 包括完成率、连续打卡天数以及各类任务的完成情况
  4. 点击"打卡日历"可以查看每日具体的打卡记录
  5. 选择特定日期可以查看当日的详细打卡情况

您还可以切换不同的时间范围（如周、月、年），查看不同时期的统计数据。

如何添加自定义打卡类型？

添加自定义打卡类型的步骤：

  1. 在任务列表页面点击"打卡类型"选项
  2. 或在创建任务页面，点击"创建新类型"链接
  3. 在打卡类型页面，点击右下角的"+"按钮
  4. 填写类型名称和描述
  5. 选择图标和颜色
  6. 点击"保存"按钮完成创建

创建自定义类型后，您可以在创建任务时选择这些类型，更好地组织和分类您的打卡任务。

如何设置打卡提醒？

设置打卡提醒的方法：

  1. 创建或编辑任务时，在"提醒时间"选项设置具体时间
  2. 或在个人中心 -> 提醒设置中，管理所有提醒
  3. 可以设置全局提醒时间和特定任务的提醒时间
  4. 还可以选择提醒方式（声音、震动等）

注意：为确保提醒正常工作，请允许应用的通知权限，并且不要将应用加入省电模式限制。
