## 创建/编辑用户表单

```
+-----------------------------------------------------------------------+
|                                                                       |
|  仪表盘 > 用户管理 > 创建用户                                           |
|                                                                       |
+-----------------------------------------------------------------------+
|                                                                       |
|  创建新用户                                                            |
|                                                                       |
+-----------------------------------------------------------------------+
|                                                                       |
|  +-------------------------------------------------------------------+|
|  |                                                                   ||
|  |  基本信息                                                          ||
|  |                                                                   ||
|  |  用户名*:  [                    ]  (必填，长度3-20个字符)           ||
|  |                                                                   ||
|  |  邮箱*:    [                    ]  (必填，用于登录和重置密码)        ||
|  |                                                                   ||
|  |  密码*:    [                    ]  (必填，长度至少8个字符)          ||
|  |                                                                   ||
|  |  确认密码*: [                    ]  (必填，与密码保持一致)           ||
|  |                                                                   ||
|  |  手机号码:  [                    ]  (选填)                         ||
|  |                                                                   ||
|  +-------------------------------------------------------------------+|
|                                                                       |
|  +-------------------------------------------------------------------+|
|  |                                                                   ||
|  |  角色与权限                                                        ||
|  |                                                                   ||
|  |  角色*:                                                            ||
|  |  ( ) 超级管理员 - 拥有所有权限，可以管理所有租户和用户                 ||
|  |  ( ) 租户管理员 - 可以管理自己租户内的用户和打卡系统                   ||
|  |  (•) 普通用户   - 无后台管理权限                                    ||
|  |                                                                   ||
|  |  所属租户*: [  选择租户  ▼  ]  (仅租户管理员和普通用户需要选择)       ||
|  |                                                                   ||
|  |  状态*:                                                            ||
|  |  (•) 正常     ( ) 禁用                                             ||
|  |                                                                   ||
|  +-------------------------------------------------------------------+|
|                                                                       |
|  [  取消  ]                                        [  提交  ]          |
|                                                                       |
+-----------------------------------------------------------------------+
```
